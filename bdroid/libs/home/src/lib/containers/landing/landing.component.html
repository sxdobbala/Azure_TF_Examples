About

A Monitoring and Tuning Application

BDroid allows tenants of the platform to drill-down, debug and report on MapR Hadoop platform metrics collected in near real-time.

BDPaaS collects the below metrics for analysis, visualization and added value for the over 100 tenants using the platform today

  • Job Metrics
  • YARN Queue Usage Statistics
  • Cluster Health Metrics
  • Yarn Individual Node Statistics
  • Yarn Metrics
  • Hive Logging
  • More

  • Cluster Usage Overview

    Help admins to plan regular cluster maintenance, snapshots etc.

  • Package inventory

    Self Service options for Tenants.

  • Metrics

    Node level Statistics about CPU, Core, Memory From MCS.

  • Storage

    Stats on number of volumes, replicas, quotas From MCS.
